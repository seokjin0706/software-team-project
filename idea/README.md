</p>

# ID6. 검토
<p>
  
1. Idea Development에서 발굴된 문제에 Idea Engineering 절차를 잘 적용했다.
  
2. 해결된 상태인 "배달비를 여러 사람이 함께 부담할 수 있어서 배달비 부담을 줄일 수 있다."를 해결하기 적합한 아이디어.
  
3. 추가 및 수정 등을 통해 핵심 아이디어를 보완했다.
  
4. 카테고리 별로 투표와 토론을 통하여 범위를 좁히고, 핵심 아이디어를 잘 도출해냈다.

</p>
